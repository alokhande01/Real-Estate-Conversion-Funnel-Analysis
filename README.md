## Real Estate Conversion Funnel Analysis (GA4)

 ## Project Overview
This project focuses on analyzing user behavior and lead conversion performance for a real estate investment platform using Google Analytics 4 (GA4).

The goal was to identify drop-off points in the user journey and evaluate traffic source performance to improve overall lead conversion rates.

 ## Objective
 
- Track lead submissions using GA4 custom events
- Build a multi-step conversion funnel
- Identify drop-offs in the user journey
- Compare traffic source performance
- Provide actionable optimization insights

 ## Tools & Technologies Used
 
- Google Analytics 4 (GA4)
- Funnel Exploration
- Event Tracking
- Traffic Source Segmentation
- Conversion Rate Analysis

 ## Funnel Structure

**Step 1:** Landing Page (page_view)  
**Step 2:** Form Interaction (form_start)  
**Step 3:** Form Submission (form_submit)  

 ## Key Findings

- ~90% drop-off between Landing Page and Form Start
- ~92% drop-off between Form Start and Form Submission
- Direct traffic showed the highest conversion rate (~4%)
- Organic traffic had significantly lower conversion (~1-2%)
- Email, SMS & Referral traffic contribution was minimal
- OTP-based form likely introduced friction in final submission

 ## Traffic Source Analysis

Traffic was segmented using:
- Organic Search
- Direct Traffic
- Referral & Affiliates
- Email, SMS & Push Notifications

Direct traffic demonstrated stronger engagement and higher form completion rates compared to other channels.

 ## Insights & Recommendations

- Improve CTA visibility on landing pages
- Optimize landing page messaging for SEO intent alignment
- Reduce form friction (simplify OTP flow)
- Strengthen trust signals before form interaction
- Improve campaign attribution for Email/SMS traffic

 ## Business Impact

This analysis provided data-driven insights to:
- Identify key conversion bottlenecks
- Improve lead generation efficiency
- Support product and growth optimization decisions
